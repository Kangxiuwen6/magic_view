时间预言魔术 - 视觉欺骗版
一款基于 HTML/CSS/JavaScript 实现的趣味魔术网页应用，通过视觉欺骗和数字计算技巧，营造出 "精准预言时间" 的魔术效果，适合线下互动表演使用。
🎮 演示效果
三位观众依次输入数字，系统通过视觉欺骗替换第三位观众的输入数字，最终三个数字的总和恰好等于当前的时间（月 + 日 + 时 + 分组合的数字），营造出 "预言时间" 的神奇效果。
📋 功能特点
🌟 视觉特效：包含故障抖动 (Glitch) 动画、渐变背景、按钮悬停效果
🎭 视觉欺骗：巧妙替换观众输入的数字，实现魔术效果
📱 响应式设计：适配不同屏幕尺寸
⌛ 时间计算：自动获取当前时间并计算对应的魔术数字
💻 纯前端实现：无需后端，直接运行 HTML 文件即可
🚀 快速magic.htmlc.html` 文件保存到本地
直接用浏览器打开该文件即可运行
按照页面提示进行互动操作
🎪 使用说明
表演流程
第一步：第一位观众输入任意数字，点击确定
第二步：第二位观众输入任意数字，点击确定
第三步：第三位观众输入任意数字（让观众记住自己输入的数字），点击确认
第四步：系统先显示观众输入的数字（建立信任），点击 "揭晓最终真相"
最终效果：展示三个数字的总和，恰好等于当前的时间（月日时分组合的数字）
魔术原理
系统自动获取当前时间并转换为数字（如：12 月 25 日 19 点 30 分 → 12251930）
用时间数字减去前两位观众输入的数字之和，得到第三个 "魔术数字"
通过视觉欺骗替换第三位观众的输入，最终总和恰好等于当前时间

暗黑渐变背景：营造神秘魔术氛围
粉紫色强调色：突出交互元素

修改视觉样式
颜色：修改 CSS 中的颜色值（如#ff66cc为其他颜色）

字体：替换font-family为其他字体
尺寸：修改各类元素的font-size、padding、margin等属性
